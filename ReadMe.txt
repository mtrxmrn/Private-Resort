How to import database, 

1. go to localhost/phpMyAdmin in web browser
2. In left side, click "New"
3. Create database name "jimsresort". This is case sensitive.
4. In navigation bar, Click "Import".
5. Drag/Choose the file from the folder xampp/htdocs/Private-Resort/Database/users.sql
6. All done and good to go
